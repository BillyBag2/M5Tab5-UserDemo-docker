# M5Tab5-UserDemo-docker

Build M5Tab5-UserDemo in a docker container.

## Instructions

Run the following to build the demo.

```bash
docker_build.sh
```

Run the following to extract executables to `./output`

```bash
./docker_extract.sh
```
